# 📚 课表侠

一款专为教师和培训机构设计的课程管理工具。支持多教师数据隔离、分钟级排课、签到管理、课时费统计，免费使用。

## 功能概览

| 功能 | 说明 |
|------|------|
| 🏠 **门户页面** | 项目介绍、功能特性展示、快速入口 |
| 👤 **邮箱注册** | 用户通过邮箱注册，管理员审核后生效 |
| 📅 **月历视图** | 按月查看所有课程安排，快速跳转到任意日期 |
| ⏰ **课程时间轴** | 06:00 到 23:00 分钟级时间轴，精确管理每节课 |
| 📊 **周课程表** | 按星期 + 小时聚合展示当月课程安排，跨小时课程自动合并 |
| 🔄 **每周重复** | 固定课程一键设置每周重复，自动生成未来一年的课程 |
| ✅ **签到管理** | 记录每节课的学生到课情况，关联课时费计算 |
| 📈 **数据统计** | 按周/月/年统计上课时长、应收费用、实收费用 |
| 🔍 **搜索筛选** | 按学生姓名、年级、签到状态模糊搜索，支持分页 |
| 🔒 **数据隔离** | 多教师账号独立管理，每位老师只能看到自己的课程数据 |
| 👑 **管理员** | admin 可查看所有教师的数据，管理教师账号 |
| 🛡️ **安全防护** | 登录频率限制、注册频率限制、蜜罐防机器人 |

## 技术栈

- **前端**: Vue 3 + Element Plus + Axios + Vite
- **后端**: Node.js + Express + SQLite3

## 快速启动

```bash
# 启动后端（端口 3002）
cd backend && npm install && node server.js
```

```bash
# 新开一个终端，启动前端开发服务器（端口 5173，支持热更新）
cd frontend && npm install && npm run dev
```

浏览器打开 **http://localhost:5173**（开发模式）或 **http://localhost:3002**（生产模式）

> 默认管理员账号：admin / admin123

## 一键启动

```bash
npm run dev     # 同时启动后端 + 前端开发服务器
npm run stop    # 停止所有服务
```

## 教师管理

```bash
cd backend
node seed.js list                                       # 查看所有教师
node seed.js add "姓名" "用户名" "密码"                   # 添加教师
node seed.js delete <id>                                 # 删除教师
node seed.js reset-password "用户名" "新密码"             # 重置密码
```

## 页面路由

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 门户页 | 项目介绍，无需登录 |
| `/login` | 登录页 | 用户名/密码登录 |
| `/register` | 注册页 | 邮箱注册（管理员审核） |
| `/calendar` | 月历页 | 课程安排概览（需登录） |
| `/day/:date` | 日详情页 | 时间轴视图（需登录） |
| `/statistics` | 统计页 | 数据搜索与统计（需登录） |
| `/admin/users` | 用户管理 | 管理员管理教师账号 |

## 安全特性

- 密码 SHA256 哈希存储，不存明文
- 随机 64 位 token 认证，登出即失效
- 登录频率限制（同一 IP 10 次/15 分钟）
- 注册频率限制（同一 IP 3 次/小时）
- 蜜罐字段 + 提交时间戳检测（防机器人批量注册）
- 数据库文件权限保护（chmod 600）
- HTTPS 加密传输（需配置域名证书）
- 每日自动备份数据库（crontab）
